Recorder Conversion Lab

Static fixture pages for validating FE recorder JSON to Agent Playwright script conversion.

How To Use Basic Interactions Selector Priority Edge Cases

How To Use This Lab

  1. Open one lab page in the browser.
  2. Start the FE recorder.
  3. Perform the listed interactions in order.
  4. Save the captured recorder JSON.
  5. Let Agent convert the JSON into Playwright script.
  6. Upload the JSON and script into the Json-To-Script-Analyser.
  7. Review the Verb Summary, Action Mapping, and Selector Priority tabs.

Basic Interactions

Inputs, checkboxes, radio buttons, dropdowns, buttons, links, double click, right click, hover, keyboard, file upload, date, color, number, range, contenteditable, alert, confirm, and scroll.

Selector Priority

Elements designed around the expected fallback order: role + name, label, test id, text, css id, stable ancestor, css class, xpath, and tag fallback.

Open Selector Page

Edge Cases

Duplicate labels, duplicate text, dynamic ids, delayed elements, hidden elements, iframe interactions, long text, and special characters.

Open Edge Page

Recommended Recording Strategy

Record one page at a time first. After each recording, run the analyser. Once each page is clean, record a larger end-to-end flow across multiple pages.

The selector priority page is the most important one for proving whether a failure is an Agent conversion issue or a FE JSON issue.
