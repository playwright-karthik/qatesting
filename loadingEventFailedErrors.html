LoadingEventFailedErrors

This page is meant to trigger various net::ERR_* loading failures.

  • CSP font-src 'none' → net::ERR_BLOCKED_BY_CSP
  • Mixed content script → net::ERR_BLOCKED_BY_CLIENT
  • Invalid SRI hash → net::ERR_SRI_SIGNATURE_MISMATCH
  • Blocked iframe (COEP/COOP) → net::ERR_INVALID_RESPONSE
  • Cross-origin image (CORP) → net::ERR_BLOCKED_BY_CORP
  • 404 Subresource → net::ERR_FAILED
  • Script with wrong MIME (Content-Type) → net::ERR_BLOCKED_BY_RESPONSE
